对SM4算法进行优化：

T-table优化是密码算法中最基础也最有效的优化手段之一，其核心思想是空间换时间：
原理：预计算S盒替换和线性变换L的组合结果，存储在4个256项的表中，每个表对应32位字中的一个字节位置
优势是避免在加密过程中重复执行相同的S盒查找和线性变换计算，将多次操作简化为一次查表操作
